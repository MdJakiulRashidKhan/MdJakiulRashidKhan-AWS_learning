# AWS SAA-C03 Learning Journey – Md. Jakiul Rashid Khan

## About This Repository
This repository is a **personal AWS study journal** for the **AWS Certified Solutions Architect – Associate (SAA-C03)** exam.  
It contains **daily notes, hands-on exercises, diagrams, and practical tips** to reinforce learning. The content is designed to be **easy to revise, clear, and applicable** in real-world scenarios.

---

## What’s Inside
- **Daily Notes:** Organized in folders (`Day-01/`, `Day-02/`, etc.) for step-by-step learning.  
- **Concept Summaries:** Key definitions, examples, and core ideas for quick reference.  
- **Hands-On Labs:** Practical exercises covering VPC, EC2, S3, IAM, Route Tables, and more.  
- **Screenshots & Diagrams:** Visual guides to understand AWS networking and architecture.  
- **Tips & Insights:** Personal notes, best practices, and learning reminders.  

---

## How to Use
1. Start with the earliest day folder and follow sequentially.  
2. Perform hands-on exercises for each service to reinforce learning.  
3. Use this repository as a **revision resource** before practice exams.  
4. You may fork or adapt this repository for your own study workflow.

## Progress & Status
- Continuously updated as I progress through the AWS SAA-C03 syllabus.
